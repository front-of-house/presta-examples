# basic

Pretty much as basic as it gets.

### Development

```
npx presta watch Home.js
```

Or for convenience:

```
npm start
```

### Production

```
npx presta build Home.js
```

Or for convenience:

```
npm run build
```

And to serve:

```
npx serve
```

Or:

```
npm run serve
```
