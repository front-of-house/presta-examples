# hybrid

A basic example of a site built partially static, partially dynamic.

Available routes out of the box:
- `/` static route
- `/about` hybrid route
- `/:slug` hybrid route
- `/dynamic/:slug` dynamic route

### Development

```
npm start
```

### Production

```
npm run build
```
